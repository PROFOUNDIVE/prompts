# Order Sense Daily Block Planner (MVP)

고급 LLM workflow를 위한 minimal&static prompt generator 및 tool 모음.

이 repo의 주요 도구는 `/order-sense/` directory에 위치한 **Order Sense Daily Block Planner (순서감각 일일 블록 계획기)**임. 이 도구는 고수준의 목표를 **Logseq (markdown editor)**에 최적화된 Block형 실행 계획으로 변환하도록 설계된 prompt generator임.

## 핵심 목표
사용자의 입력을 LLM(ChatGPT/Gemini)을 위한 구조화된 프롬프트로 변환하여, 결과물이 사용자의 Logseq Journals에 원활하게 통합되는 특정적이고 반복 가능한 구조를 따르도록 함.

## 핵심 철학
이 플래너는 다음과 같은 과정에 집중하는 **순서감각 (Order Sense)** 프레임워크에 기반을 두고 있습니다:
**인지 (Cognition) → 이해 (Understanding) → 실행 (Execution)**

전체적인 그림을 먼저 그리는 계획, 의존성을 고려한 순서 지정, 에너지 기반 작업 할당을 강조합니다.

## UX 흐름
1. **입력**: 구조화된 입력 필드(목표, 예산, 제약 조건 등)를 작성합니다.
2. **생성**: "Generate Prompt"를 클릭하여 시스템에 최적화된 프롬프트를 생성합니다.
3. **복사**: 프롬프트 또는 출력 템플릿에 대한 원클릭 복사 버튼을 사용합니다.
4. **붙여넣기**: 결과를 LLM(ChatGPT/Gemini)에 붙여넣어 포맷팅된 계획을 받습니다.

## 주요 기능
- **구조화된 입력**: 목표, 예산, 제약 조건, 결과물 증거, 에너지/스트레스 수준, 검토 항목에 대한 포괄적인 필드를 제공합니다.
- **하이브리드 프롬프트 로직**: 사용자의 선호 언어를 유지하면서 LLM의 추론 품질을 극대화하기 위해 "영어로 추론하고 한국어로 출력"하는 시스템 프롬프트 로직을 사용합니다.
- **편의성**: 전체 프롬프트와 원본 출력 템플릿 모두에 대해 원클릭 복사 버튼을 제공하며, LocalStorage를 통해 브라우저 세션 간에 입력을 저장하고 유지합니다.
- **Logseq 최적화**: Logseq 내에서 쉽게 붙여넣고 정리할 수 있도록 특화된 출력 형식을 제공합니다.

## 핵심 철학
이 플래너는 **순서 감각(Order Sense)** 프레임워크를 기반으로 하며, **인지 → 이해 → 실행** 프로세스에 중점을 둠.

먼저 큰 그림을 스케치하고, 의존성을 고려하여 작업을 순서화하며, 에너지 기반 작업을 할당하는 방식으로 계획을 수립하는 것을 강조함.

## UX Flow
1. **Input (입력)**: 사용자가 구조화된 입력 필드(목표, 시간 예산, 제약 조건 등)를 작성
2. **Generate (생성)**: “프롬프트 생성”을 클릭하여 최적화된 프롬프트를 생성
3. **Copy (복사)**: 프롬프트 또는 출력 포맷 템플릿만 원클릭 복사 버튼으로 복사
4. **Paste (붙여넣기)**: 결과를 LLM(ChatGPT/Gemini)에 붙여넣어 형식화된 계획을 얻음

## 주요 기능
- **구조화된 입력**: 목표, 예산, 제약 조건, 결과물 증거, 에너지/스트레스 수준, 검토 항목에 대한 포괄적인 필드를 제공함
- **하이브리드 프롬프트 로직**: “영어로 추론하고 한국어로 출력”하는 시스템 프롬프트 로직을 사용하여 LLM 추론 품질을 극대화하면서 사용자가 선호하는 언어를 유지함
- **편의성**: 전체 프롬프트와 원본 출력 템플릿 모두에 대한 원클릭 복사 버튼을 제공하며, LocalStorage를 통해 브라우저 세션 간 입력을 저장하고 유지함
- **Logseq 최적화**: Logseq 내에서 쉽게 붙여넣기 및 서식 지정할 수 있도록 특수 출력 형식을 제공함
